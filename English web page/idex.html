Computer Programming


By Jacob Alvarado

What do computer programmers do?

Computer programmers create code and scripts that allow computer applications to run. All though mainly known for creating code, computer programmers also analyze, rewrite, correct, and revise code. From personal experience, theirs nothing more irritating for a computer programmer than writing portions of code just for it to not work. Any game, web page, and applications that you can think of was coded by computer programmers.

Similar to speaking languages, theirs also many different languages within coding! Although having similar ways of writing code, each language has its own methods to do so. Each language usually has its own purposes, for example, JavaScirpt is used in webpages and Java is used in game development.
Some of the most well-known coding languages are:

  • Java Scirpt
  • Python
  • C++
  • Java
  • C
  • PHP
  • SQL
  • Ruby

Whats skills are required?

The most noticeable skills for a computer programmer is to know how to write and execute code. The coding language each computer programmer should focus on depends on which field of computer programming they’d like to follow. Other then just coding, it's recommended to also know the in and outs of computer functionality.
Some other important skills computer programmers should come equip with is:

  • Problem solving: When coding anything, you are most likely trying to solve a problem, so problem solving skills are heavily required

  • Logical thinking: Working with a computer means you need to instruct the computer with exact tasks. Thinking logically is key to make your code work.

  • Cooperative and verbal: Computer programmers tend to work in teams, so being able to cooaporate with your teammates and express your ideas is important.

  • Math: Arithmetic should be well known. Skill levels from basic all the way to Algebra should be sufficient



Profession context

Computer programmers most commonly work in offices locations within information technology. Depending on their interest though, computer programmers can work anywhere from healthcare, government, as well as education. Some computer programmers can even work remotely from home.

If you've wondered how computer programmers even write code in the first place, they use applications called "code editors." Code editors allow programmers to write and execute the code. Many code editors exist, but one fo the most used ones is called "Visual Studio code."




Preparation and requirements

The most noticeable requirement for computer programmers are their education requirements. Although not every job in computer programming requires a degree, the standard requirement is a bachelors degree in computer science.

Other then that, its also recommended to have knowledge in Engineering, Arts and humanities, and business.

Although a profession of its own within the computer programming tree, a good place to start is web development.

Web development introduces us into the field of coding, as well as learning Java script.

The average salary of a computer programmer here in Texas is $49.35 hourly, ranging anywhere from $24.73-$70.22
